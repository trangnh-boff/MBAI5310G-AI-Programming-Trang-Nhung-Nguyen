# Early Loan Repayment Classification Project

## Project Overview

This project builds a machine learning pipeline to predict whether a customer will repay a loan early or not.

The target variable is:

**Early_Repayment**


## Dataset

The dataset contains customer financial information such as:

* Age
* Annual Income
* Credit Score
* Loan Amount
* Loan Term
* Interest Rate
* Savings Balance
* Missed Payments
* Employment Status
* Loan Type
* Automatic Payment

## Machine Learning Steps

This project includes:

1. Understanding the business problem
2. Data preparation
3. Data cleaning
4. Feature and target definition
5. Train/test split
6. Preprocessing
7. Logistic Regression model
8. SVM model
9. Model evaluation
10. Model comparison

## Models Used

Two models were trained:

* Logistic Regression
* Support Vector Machine (SVM)

## Evaluation Metrics

The models were evaluated using:

* Accuracy
* Precision
* Recall
* F1-score
* Confusion matrix

## Final Results

### Logistic Regression

* Accuracy: 73.68%
* Precision: 65.52%
* Recall: 65.52%
* F1-score: 65.52%

### SVM

* Accuracy: 67.11%
* Precision: 57.14%
* Recall: 55.17%
* F1-score: 56.14%

Logistic Regression performed better overall.

## Responsible AI Reflection

Machine learning models should be used as decision-support tools.

Predictions may not always be correct, so businesses should still use human judgment, fairness considerations, and ethical decision-making when using customer financial data.
