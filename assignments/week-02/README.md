# MBAI 5310G - Week 2 ML Pipeline
## Project Purpose
This project builds a simple machine learning pipeline using Logistic Regression.
## Dataset Used
Finance Early Loan Repayment Prediction Dataset
## Business Problem
A financial institution wants to predict which customers are likely to repay their loans early.
## Main Steps
- Load and inspect the dataset
- Clean the data
- Define features and target
- Apply preprocessing
- Split the data
- Train a Logistic Regression model
- Evaluate the model
## Model Limitation
Logistic Regression is a simple baseline model and may not capture complex financial behaviour patterns.
