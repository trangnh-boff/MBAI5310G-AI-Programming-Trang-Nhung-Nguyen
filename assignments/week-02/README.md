# Week 2 ML Pipeline
## Business Problem
A financial institution wants to predict which customers are likely to repay their loans early.
## Dataset
Finance Early Loan Repayment Prediction Dataset
## Machine Learning Model
Logistic Regression
## Main Steps
- Data inspection
- Data cleaning
- Preprocessing
- Train/test split
- Logistic Regression
- Model evaluation
## Limitation
Logistic Regression is a simple baseline model and may not capture complex financial behaviour patterns.
